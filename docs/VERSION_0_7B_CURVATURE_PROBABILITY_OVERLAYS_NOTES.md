# RealMathUniverse v0.7B Curvature + Probability Overlays

## What changed

v0.7B adds static GPU-rendered visual field aids:

```text
1. Curvature rings
2. Probability halo
```

## Curvature rings

Curvature rings are warped concentric rings drawn as an overlay. They are meant to make field boundaries and lens-like structure more readable during live presentation.

Toggle with:

```text
O
```

## Probability halo

The probability halo is a low-alpha golden-angle point distribution. It is meant to create a visible probabilistic boundary layer around the active cloud.

Toggle with:

```text
G
```

## Notes

These overlays are visual and diagnostic. The real field-layer transform from v0.7A remains active and independent.
